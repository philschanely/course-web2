<p class="t-lead">LearnUXD brings you this course on how to take the next steps to quickly but effectively build out web pages and components with advanced CSS technique and an eye for usability and accessibility.</p>

By the end of this course students will be able to:

- Communicate professionally using knowledge of web design terms.
- Work fluently with modern technologies and best practices to develop and enhance web pages for usability, searchability and accessibility.
- Locate and use various resources to further develop knowledge and skills for web design.
- Apply grid design, web aesthetics and appropriate style to develop advanced, visually appealing, and usable web pages, web sites, and content navigation in a variety of genres.
